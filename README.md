# Smart E‑Waste Collection and Management System

Final Year Project

A web-based platform to manage e-waste collection, pickup, and disposal through dedicated **User**, **Collector**, and **Admin** workflows — making responsible e-waste disposal easier to schedule, track, and manage.

## Live Demo


## Features

- Role-based portals for Users, Collectors, and Admins
- JWT-based authentication
- E-waste pickup scheduling and request tracking
- Collector assignment and status updates
- Admin dashboard for managing users, approving collectors, and pickups

## Tech Stack

- **Frontend:** React (Vite), Tailwind CSS, Axios, React Router
- **Backend:** Node.js / Express
- **Database:** MySQL
